Instructions to Run Code:
    Requirements:
        - Python 3.11
        - matplotlib
    Commands:
        - pip3 install matplotlib
        - python3 main.py

    You will be prompted:
        - Enter 1 for UserTestMode or 2 for ScatterPlotMode
        - If UserTestMode:
            - Enter value for k
            - Enter value for a
            - Enter value for n
            - Enter list size (10-100, increment 10)
        - If ScatterPlotMode:
            - Multiple plots will be generated.
                
    Note: Close each matplotlib window to see the next plot.
